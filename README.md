# Ashley Heart U. Golloso — Data Scientist Portfolio

A personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
npm install
npm run dev
```

## Deployment

Build for production:

```sh
npm run build
```
